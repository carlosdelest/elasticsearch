About the files in this directory

- `cloud-idp.crt` and `cloud-idp.key` were generated with
   ```
   bin/elasticsearch-certutil cert --self-signed --pem --out "${PWD}/idp-cert.zip" --name "cloud-idp" --days 5000 --keysize 1024
   ```

- `metadata.xml` was generated by hand, using `cloud-idp.crt`
